Declaring variables

You can declare a variable in different ways

int x = 3

This syntax is used to declare whole numbers

assigning a variable without a keyword is wrong

x = 7

This generates an error.. don't use this syntax

you can assign values and specify if the element is a floating point number or a string as such:

float x = 9.65

for floating point

string x = "Hello world"

for declaring strings
